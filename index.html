    Designed for accessing websites in challenging network environments, censorship.no (CENO) is a mobile web browser that allows you to browse the Web without restriction, circumventing any network filtering. The underlying architecture offers peer-to-peer routing and distributed caching, using 'darknet' infrastructure to avoid network censorship and bring the Web to your browser.

    The CENO project presents a new generation of censorship circumvention possibilities. Our project is built in support of Articles 18, 19 and 20 of the Universal Declaration of Human Rights.
