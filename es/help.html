Traducciones

¡Ayuda a traducir la aplicación del Navegador CENO, la extensión web, el manual de usuario y el sitio web!

Para empezar:

  • Crea una cuenta en Weblate o inicia sesión con tu ID de Github o Gitlab. Nota: La creación de una cuenta no es requerida, pero en este caso solamente serás capaz de sugerir traducciones.
  • Visit the Censorship.no project page - which lists all components and their strings (including Android app strings, the web browser extension, the User Manual and the website).
  • En la pestaña 'Languages' de la página del proyecto Censorship.no, elige el idioma sonre el que quieres trabajar, o -- en el caso de que aún no exista -- inicia uno nuevo seleccionando un componente desde la página principal del proyecto y haciendo clic en el botón "Start a new translation".
  • Por favor contacta al equipo CENO a través del formulario de contacto de abajo si estás interesado en convertirte en revisor para las traducciones en tu(s) idioma(s).
  • Por más información acerca de cómo usar Weblate, mira la documentación de Weblate.


Source Code

  • Componentes del navegador: constructor del paquete, ramal Firefox, extensiones incluidas, fuente para configuración de extensiones

Puede que también estés interesado en la (ya no mantenida) versión previa de CENO, basada en la red anónima de compartición de archivos y publicación de contenidos Freenet. Otros repositorios inactivos relacionados con el proyecto pueden ser encontrados en el archivo de Censorship.no!.
