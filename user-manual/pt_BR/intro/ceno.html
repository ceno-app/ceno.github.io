Censorship.no! Manual de Usuário

O que é o Navegador Ceno?

Ceno (abreviação para Censorship.no!) é um navegador Web para dispositivos Android (tais como smartphones e tablets) que usa uma nova abordagem para contornar a censura na Internet e compartilhar conteúdo recuperado entre todos os usuários em estilo par-a-par (P2P). Isso reduz a dependência generalizada nas trocas da rede internacional e permite que conteúdo Web frequentemente requisitado persista na rede mesmo durante eventos massivos de filtragem e limitação.

O que diferencia o Ceno da maioria das outras iniciativas de contorno é que os usuários podem continuar a compartilhar conteúdo Web mesmo quando não há conectividade (ou ela está extremamente limitada) ao longo das fronteiras nacionais. Ceno foi, assim, desenvolvido em antecipação às filtragens agressivas da Internet e ao estabelecimento de intranets nacionais para deixar nações de fora da Web.

O Navegador Ceno é construído em uma versão adaptada do Firefox para Android, um navegador gratuito, livre e de código aberto popular, moderno, rico em funcionalidades e seguro. Ceno estende o Firefox com a Ouinet, a tecnologia subjacente que permite ao navegador compartilhar conteúdo entre dispositivos (descrito nas próximas seções).

Quem desenvolve o Ceno?

The Censorship.no! project is run by eQualitie in support of Articles 18, 19 and 20 of the [Universal Declaration of Human Rights][Universal Declaration of Human Rights]. Ceno and related technologies are developed as Free/Libre/Open Source software (project source), allowing anyone to use, study, share and enhance it. Please contact Cenoers@equalit.ie in case of doubt or for further information.

É feito para quem?

Ceno é feito para todas as pessoas, mas é particularmente conveniente para as interessadas em conteúdo Web que está censurado na sua rede e para aquelas habitando países onde a conectividade com a Internet global é errática, não confiável ou cara. Ele incentiva e permite o compartilhamento de conteúdo Web entre os usuários, criando uma rede descentralizada de pares que se ajudam mutuamente.

Você não precisa ser um usuário de computador avançado ou mesmo entender como redes P2P operam para usar o Ceno. A experiência do usuário é similar a operar um navegador Web comum (funcionalidades específicas do Ceno são em grande parte invisíveis).

Ceno pode, contudo, gerar mais tráfego na Internet que o seu navegador Web usual - pois ele precisa deixar outros usuários saberem qual conteúdo Web está compartilhando com a rede e possivelmente entregar esse conteúdo para aqueles que o solicitam. Desse modo, o Ceno depende de conectividade moderadamente boa dentro do país. Recomendamos usar o Ceno quando houver conexão WiFi, não só para evitar atingir os limites de uso de dados móveis (caso os tenha), como para aumentar as chances de conseguir entregar conteúdo Web para outros usuários.

Aviso: Ceno não é uma ferramenta de anonimato. Na verdade, usar o Ceno pode permitir que outras pessoas saibam se você acessou ou está compartilhando determinado conteúdo Web. Por favor, tenha bastante cautela ao considerar os riscos que você pode assumir ao usar essa ferramenta. Veja as seções sobre navegação pública versus navegação privativa e riscos para mais informações.
